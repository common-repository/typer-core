== Description ==

Typer Core adds features to Typer theme like post settings, customizer sections and content import ability.
This plugin requires the Typer theme to be installed.

== Installation ==

Manual installation method requires downloading the Typer Core plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended

= Updating =

Updates should work smoothly, but we still recommend you back up your site.
